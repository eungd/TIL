# Markdown 



### Markdown 이란?

✔텍스트 기반의 가벼운 ''마크업 언어''

✔최소한의 문법으로 다양하게, 다양한 곳에 깨짐없이 연동가능



✏확장자 `.md ` 사용한 것이 markdown문법 사용한 파일.

✏따라서 `.txt` 파일 생성 후 `.md` 로 확장자를 바꾸는 것으로 파일 변경 가능.
