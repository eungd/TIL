# Fenced Code block

✔Syntax Highlighting 적용을 위한 문법 



✏ 표현하고자 하는 코드 블록에  ``` 를 사용하여 명시하려는 언어를 특정하면 적용가능    



예를 들어 아래와 같은 코드를

{

"firstName": "John",

"lastName": "Smith",

"age": 25

}



```json
{
"firstName": "John",

"lastName": "Smith",

"age": 25

}
```



특정언어로 syntax highlighting 할 수 있음.
